
格拉摩根的野望

游戏简介：
1. 此游戏主体剧情按照P社游戏《王国风云》和《欧陆风云4》设计而来
2. 在游戏中，玩家将扮演替死鬼格拉摩根伯爵，尝试进行一场复兴罗马帝国的伟大任务。
3. 此游戏场面战争情节描写由《堂吉诃德》、《水浒传》、《三国演义》、《基督山伯爵》改编而来。
4. 此游戏的事件情节由《欧陆风云4》一些脍炙人口的event改编而来。

游戏说明：

此游戏中有四个等级，公爵，国王、皇帝和奥古斯都。玩家开始处于公爵等级，玩家可以通过发动战争、
个人决斗等方式实现地位的提升。但在此过程中也面临着诸多的危险，玩家可能会因此丧命。当玩家到达
奥古斯都的位置时，游戏结束。

此游戏中玩家会有三个选择提升自己的地位：
1. 发起战争
此种情形下玩家会依靠自己的国家进行战争，双方将从金钱，军事等角度进行比拼，
每场战争包含两次战役，当玩家的军队数全部被敌人歼灭时，游戏失败。然而如果玩家的军队没有被
地方消灭，打败仗就会面临国内的叛军。若玩家打胜仗，地位就会上升。
2. 个人决斗
此种情形下玩家会和对方进行一对一的决战，为了取胜将不择手段。包括投毒，刺杀等。玩家若在此过程中
HP降为0，游戏就会失败。
3. 休养生息
在此情形下玩家会面临各种各样的事件，包括好事和坏事，都会对玩家统治的国家和玩家本身造成影响。
